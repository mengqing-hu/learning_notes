# DNS 是什么？

**DNS（Domain Name System），域名系统**

负责把好记的域名（比如 `baidu.com`）翻译成机器能懂的 IP 地址（比如 `110.242.68.66`）。

## 1. DNS 完整工作流程

你在浏览器输入 `baidu.com`：

1. 浏览器问本地系统：你知道 [baidu.com](https://baidu.com) 的 IP 吗？
2. 不知道 → 去找 **本地 DNS 服务器**（运营商 / 路由器）
3. 本地 DNS 不知道 → 去找 **根 DNS 服务器**
4. 根服务器说：去问 `.com` 顶级服务器
5. `.com` 服务器说：去问 baidu 自己的权威 DNS
6. 权威 DNS 返回真实 IP
7. 浏览器拿到 IP，去连接服务器

整个过程一般 **几十毫秒**，你完全感觉不到。

## 2. 和 CDN 有什么关系？

**DNS 是 CDN 的 “调度员”**

当你开启 CDN 后：

- 用户访问域名
- DNS 不会返回你源站 IP
- 而是**智能返回离用户最近的 CDN 节点 IP**

这就是 CDN 能实现 **“就近访问”** 的核心原理。
