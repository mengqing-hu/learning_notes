# Agent 范式（Agent Paradigms）

**Agent = LLM 大脑 + 记忆 + 规划 + 工具 + 执行闭环**

它不再是单次问答，而是**能自主感知、决策、行动、反思**的智能体。

------

### 一、什么是 Agent 范式？

Agent 范式是**基于大模型构建自主智能体的一整套框架思路**，核心是让模型：

- 理解目标
- 维护状态与记忆
- 自主规划步骤
- 调用外部工具
- 根据反馈修正行为
- 最终完成复杂任务

一句话：**LLM 是大脑，Agent 是带手、脚、记忆、规划能力的完整智能系统。**

------

### 二、Agent 的标准通用架构

1. **Memory（记忆模块）**: 短期记忆、长期记忆、向量记忆、经验记忆
2. **Planning（规划模块）**: 任务拆解、多步推理、决策
3. **Tool Use / Action（工具 / 行动模块）**: 搜索、计算、代码、API、数据库
4. **Execution Engine（执行引擎）**: 循环控制、状态流转、异常处理
5. **Reflection / Self-Correction（反思模块）**:纠错、优化、避免循环、提升效率

------

### 三、目前主流五大 Agent 范式

#### 1. ReAct（最基础、最主流）

**Reason + Act 交替执行**

- **流程：Thought → Action → Observation → 循环**
- 特点：边想边做、小步迭代、灵活
- 优点：适配动态环境、可解释强
- 缺点：步骤多、易循环、延迟高
- 代表：ReAct、FuncCall Agent

#### 2. CoT 系（纯推理范式，无行动）

**Chain-of-Thought**

- **只思考、不行动、不调用工具**
- 适合：数学、逻辑、文本生成
- 是 Agent 的 “推理底座”，但不是完整 Agent

#### 3. Plan-and-Execute（先规划后执行）

**先生成完整计划 → 再批量执行**

- 代表：Plan-and-Solve、BabyAGI、AutoGPT 早期
- 优点：步骤清晰、调用少、效率高
- 缺点：对环境变化鲁棒性差，计划错则全崩

#### 4. Reflection / Self-Refine 范式（反思增强）

**ReAct + 反思机制**

- 每几步或失败后复盘：哪里错了？怎么改？
- 代表：Reflexion、Self-Correct、Self-Improve
- 解决：循环、幻觉、无效行动、次优路径

#### 5. Multi-Agent 范式（多智能体协作）

多个 Agent 分工合作：

- 专家分工（搜索、代码、写作、审核）
- 辩论、博弈、协作、投票
- 代表：ChatDev、MetaGPT、AutoGen、LangGraph 多节点
- 优点：复杂任务能力极强
- 缺点：延迟高、调度复杂

------

### 四、Agent 范式的核心能力（区别于普通 LLM）

1. **自主决策**：不需要每步人工引导
2. **工具使用**：能调用外部系统扩展能力
3. **状态保持**：记住目标、进度、历史
4. **环境交互**：能感知环境并根据反馈调整
5. **任务闭环**：从开始到结束自动完成
6. **自我修复**：能发现错误并修正

### 五、Agent 当前面临的核心挑战

- 上下文过长 → Token 爆炸
- 无限循环、无效行动
- 幻觉导致工具调用错误
- 延迟高、API 成本高
- 安全性、可控性不足
- 复杂任务规划能力不稳定

所有范式都围绕：**记忆、规划、工具、执行、反思**五大模块构建。

### 六、示例

##### ReAct执行流程（Thought → Action → Observation）

1. **Thought（思考）**：LLM 分析当前状态、明确目标、规划下一步行动
2. **Action（行动）**：调用工具（搜索、计算、API、数据库等）或执行操作
3. **Observation（观察）**：获取工具返回结果，更新上下文
4. **循环**：重复 1–3，直到得出最终答案或满足停止条件

**示例（问答场景）**：

> **Question**：2025 年诺贝尔物理学奖得主是谁？
>
> **Thought**：我需要搜索 2025 年诺贝尔物理学奖的最新信息。
>
> **Action**：Search["2025 Nobel Prize in Physics winners"]
>
> **Observation**：搜索结果显示：2025 年诺贝尔物理学奖授予 XXX，表彰其在 XXX 领域的贡献。
>
> **Thought**：已有明确答案，可直接回复。
>
> **Answer**：2025 年诺贝尔物理学奖得主是 XXX，以表彰其在 XXX 方面的突破性成就。
